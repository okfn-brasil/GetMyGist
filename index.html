The idea of this project is to run/write your gist on the web. Many times you need run a small javascript but need build a server or another thing for to do it, with Get my Gist you can make it very easy, only connect with github account and run your gist

How it works

You will connect with github account and Get my gist get all your gist from gist.github.com, if you have a javascript gits will show option to run this gist, else you can only edit it
